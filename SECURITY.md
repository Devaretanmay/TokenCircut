# Security

## Reporting a vulnerability

Report security vulnerabilities to the admin.

Do not file public GitHub issues for security vulnerabilities.

## Response

We aim to acknowledge reports within 48 hours and provide a fix timeline within 5 business days.

## Scope

- The Python library `tokencircuit` (src/tokencircuit/)
- The control plane dashboard (control-plane/)
- Supabase Edge Functions (supabase/functions/)

## Out of scope

- Vulnerabilities in frameworks LangGraph, CrewAI, or OpenAI SDK
- Misconfiguration of the library by end users
